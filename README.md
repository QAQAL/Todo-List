# Todo-List
一个基于Ajax的Todo List 和Weibo List

前端用 Ajax 发送 HTTP 请求到后端，后端只对数据进行增删改查，实现前后端分离；

对Ajax进行独立封装，以函数方式调用；

设计后端 API 接口，无需刷新页面即可修改网页内容；
